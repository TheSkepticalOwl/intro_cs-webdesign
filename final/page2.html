HTML

HTML is the basis of the web. It uses markup to define elements that can be modified using CSS and JacaScript.

CSS

CSS is used to change how HTML elements look and feel. It uses a simple structure designed to make it easy for programmers and web designers to use.

JavaScript

JavaScript is used for creating games as well as improving some user experience in websites. Although optional, it is usually used on websites to change different elements of a page and make it easier to create great user experience.
